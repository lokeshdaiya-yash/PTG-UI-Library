# Alert Component

---

The alert method displays an alert with a message .

Alerts are available for any length of text, as well as an optional dismiss button.

The alert method is used when you want information to come through to the user.

We can import the alert using the following importation :

```
    import { AlertModule } from ' @ptg-ui/angular'
```

Types of controls supported by button:

- message
- isDismissible

## Quick Start

we can create a alert using the following HTML Tag:

```
<ptg-ui-alert message="Hello" [isDismissible]="true"></ptg-ui-alert>

```

Alert messages can be used to notify the user about something special.
