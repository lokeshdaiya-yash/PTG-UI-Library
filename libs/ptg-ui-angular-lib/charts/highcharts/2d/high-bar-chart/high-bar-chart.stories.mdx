 # HighBarChart Component

 ---
 Bar Chart is the most common chart type that is used to compare different categories of data like frequency, count, total, or average displayed in vertical bars. 

 We can import the HighBarChart Component using the following importation : 

```
    import { HighBarChartComponent } from ' @ptg-ui/angular'
```    


 ## Quick Start
 
 we can create a highBarChart using the following tag: 

```
<ptg-ui-high-bar-chart [data]="data"></ptg-ui-high-bar-chart>

```
