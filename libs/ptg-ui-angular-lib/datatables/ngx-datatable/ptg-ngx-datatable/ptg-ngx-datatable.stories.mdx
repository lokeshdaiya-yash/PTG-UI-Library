 # PtgNgxDatatable Component

 ---

ngx-datatable is a Angular component for presenting large and complex data.

It has all the features you would expect from any other table but in a light package with no external dependencies. 

The table was designed to be extremely flexible and light; it doesn't make any assumptions about your data .


 We can import the PtgNgxDatatable using the following importation : 

```
    import { PtgNgxDatatableModule } from ' @ptg-ui/angular'
```    

 Types of controls supported by PtgNgxDatatable:

 - scrollbarV
 - scrollbarH
 - headerHeight
 - footerHeight
 - rowHeight
 - loaderShowWhen
 - offset
 - limit
 - externalSorting
 - rowClass
 - showFiter
 - showActionButton
 - actionButtonLabel
 - actionButtonHeaderLabel


 
 ## Quick Start
 
 we can create a PtgNgxDatatable using the following HTML Tag: 


```
<ptg-ui-ptg-ngx-datatable
  [scrollbarV]="false"
  [scrollbarH]="false"
  [headerHeight]="50"
  [footerHeight]="100"
  [rows]="rows"
  [columns]="columns"
  rowHeight="auto"
  [loaderShowWhen]="false"
  [offset]="0"
  [limit]="4"
  [externalSorting]="false"
  rowClass=""
  [showFiter]="true"
  [selected]="selected"
  [showActionButton]="false"
  actionButtonLabel="Click Here"
  actionButtonHeaderLabel="Action"
></ptg-ui-ptg-ngx-datatable>

```


Following are some controls with their explanation :


1] scrollbarV :

scrollbarV is used to hide and show the vertical scrollbar.

scrollbarV control have two values : 

a] True - scrollbar is hide and also data is hide.

b] False - scrollbar and data is visible.

2] scrollbarH : 

scrollbarH is used to hide and show the horizontal scrollbar.

scrollbarH control have two values : 

a] True - scrollbar is hide .

b] False - scrollbar is visible.

3] headerHeight : 

we can adjust the height of header using the header height control.

by default , the header height is 50px.

4] footerHeight : 

we can adjust the height of footer using the footer height control.

5] rowHeight :

<Canvas><Story id="component-ptgngxdatatablecomponent--row-height" /></Canvas>


6] loaderShowWhen : 

Basically Loader is used to show the loading state of data in application.

loaderShowWhen control have two values :

a] True - loader is shown.

b] False - loader is not visible.


<Canvas><Story id="component-ptgngxdatatablecomponent--loader-show-when" /></Canvas>



7] limit : 


limit control is used to set the number of rows on page.

<Canvas><Story id="component-ptgngxdatatablecomponent--limit" /></Canvas>


8] externalSorting :

By default, the sorting order will be as ascending or descending or none. 

When first click a column Athlete it sorts the column in ascending. 

Again click the same column Athlete, it will sort the column in descending.

Same for numbers also in sorting. All the columns sorting in ascending and descending order.

externalSorting control have two values :

a] True - we can do the sorting for the table.

b] False - we can not do the sorting for the table.

<Canvas><Story id="component-ptgngxdatatablecomponent--external-sorting" /></Canvas>


9] rowClass :

The rowClass  is used to specify a class for an  element.

The rowClass is often used to point to a class name in a style sheet. It can also be used by a other languages and manipulate elements with the specific class name.

Multiple HTML elements can share the same class.

10] showFiter : 

showFiter control is used to hide and show the filter.

showFiter control have two values -

a] True - showFiter is visible.

b] False - showFiter is not visible.

<Canvas><Story id="component-ptgngxdatatablecomponent--show-fiter" /></Canvas>



11] showActionButton :
 
showActionButton control adds the new column of action.

Under that column it will show the Click here button.

showActionButton control have two values -

a] True - add new column of button.

b] False - remains same.
