 # MatDatatable Component

 ---

The Material Data Table component is a generic component for displaying tabulated data. 

Although we can easily give it a Material Design look and feel, this is actually not mandatory.

In fact, we can give the Angular Material Data table an alternative UI design if needed. 


We can import the datatable using the following importation : 

```
    import { MatDatatableModule } from ' @ptg-ui/angular'
```    


Types of controls supported by datatable:

- emptyStateMessage
- totalCount
- limit
- showFiter


 
 ## Quick Start
 
 we can create a datatable using the following HTML Tag: 


```
<ptg-ui-mat-datatable
  [data]="data"
  [columns]="columns"
  emptyStateMessage="No data matching the filter"
  [totalCount]="20"
  [limit]="10"
  [showFiter]="true"
></ptg-ui-mat-datatable>

```


Following are some controls with their explanation :

1] emptyStateMessage - 

An empty state message appears after someone has completed a task or workflow, or has cleared all data associated with certain functionality.

<Canvas><Story id="component-matdatatablecomponent--empty-state-message"/></Canvas>


2] totalCount - 

We can set the length of the pagination using totalCount.

After the data is completed it shows the error message for other pages.


<Canvas><Story id="component-matdatatablecomponent--total-count"/></Canvas>


3] limit - 

Limit allows you to set the number of rows on the page.

<Canvas><Story id="component-matdatatablecomponent--limit" /></Canvas>   


4] showFiter - 

showFiter control is used to hide and show the filter.

showFiter control have two values -

a] True - Fiter is visible.

b] False - Fiter is not visible.
