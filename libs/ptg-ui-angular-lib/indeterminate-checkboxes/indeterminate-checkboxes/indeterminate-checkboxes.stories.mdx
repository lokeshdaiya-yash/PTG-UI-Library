 # Indeterminate-Checkboxes Component

  ---
Indeterminate checkboxes are displayed in a state which is distinct both from being checked or being unchecked. 

They are commonly used in hierarchical checkboxes to indicate that only some of the checkbox's descendants are checked.


```
    import { IndeterminateCheckboxesComponent } from ' @ptg-ui/angular'
```    


 ## Quick Start
 
 we can create a indeterminate-checkboxes using the following HTML Tag: 


```
<ptg-ui-indeterminate-checkboxes
  [checkBoxList]="checkBoxList"
></ptg-ui-indeterminate-checkboxes>

```
