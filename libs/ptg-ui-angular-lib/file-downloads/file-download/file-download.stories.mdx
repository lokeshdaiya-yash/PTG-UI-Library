# File-Download Component

---

File download component is used to download the files.

we can select the type of file from dropdown. 

Following are some types of files :

- PDF
- Excel
- Word 
- Download PNG


We can import the FileDownloadComponent using the following importation : 

```
    import { FileDownloadsModule } from ' @ptg-ui/angular'
```    


 ## Quick Start
 
 we can download the file using following HTML Tag: 


```
<ptg-ui-file-download [tableData]="tableData"></ptg-ui-file-download>
```
